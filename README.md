##Newsoap: Simplify your news stream

#####Newsoap is a filter for news articles that allows you to focus on the news topics you want.

[Heroku](https://newsoap.herokuapp.com)

Built with: 
* Ruby on Rails
* Javascript
* Devise gem
* Brakeman gem
* HTTParty gem
* Skrollr gem

Utlizing the NPR API.
